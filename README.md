# 🎨 로컬 이미지 생성·편집 스튜디오

**자연어로 이미지를 만들고·고치고·합치는 프로그램입니다.**
내 ChatGPT 계정으로 로그인만 하면 됩니다. (별도 결제·API 키 설정 없음 — 본인 ChatGPT 구독 사용)

---

## 🚀 이렇게 쓰세요 (탐색기에서 더블클릭만!)

> CMD(검은 창) 몰라도 됩니다. **폴더 열고 더블클릭 2번**이면 끝이에요.

### 처음 한 번만 — 아래 2개 설치 (각각 다운로드 후 "다음"만 누르면 됨)
1. **Python** → https://www.python.org/downloads/ → 설치 첫 화면에서 **`Add Python to PATH` 체크** 후 설치
2. **Node.js** → https://nodejs.org/ → 그냥 "다음"만 눌러 설치

### 그다음 (매번 이렇게)
1. 이 폴더를 **파일 탐색기**로 엽니다.
2. **`setup.bat`** 더블클릭 → 자동 설치가 진행되고, 처음이면 **브라우저가 열려 ChatGPT 로그인** 창이 뜹니다. → 본인 ChatGPT 계정으로 로그인.
3. **`run.bat`** 더블클릭 → 잠시 뒤 **브라우저에 앱이 자동으로 열립니다.** 🎉

> 💡 다음부터는 **`run.bat` 더블클릭 한 번**이면 바로 켜집니다. (setup은 처음 한 번만)
> 💡 끌 때는 같이 떠 있는 검은 창을 닫으면 됩니다.

---

## ✏️ 사용법

1. **만들기** — 아래 입력창에 원하는 그림을 설명하고 Enter
   - 예) `미니멀 커피 브랜드 로고, 텍스트 "DAILY", 둥근 산세리프, 흰 배경`
2. **고치기** — 만든 이미지를 클릭 → 오른쪽에서 자연어로 수정
   - 예) `배경을 베이지색으로 바꾸고 로고는 그대로`
   - 마우스로 영역을 칠해서 그 부분만 고칠 수도 있어요
3. **합치기** — 입력창 옆 **＋** 버튼으로 여러 이미지를 골라 한 장으로 합성
4. 마음에 드는 그림은 **다운로드** 버튼으로 저장

왼쪽 **사이드바**에서 리스트(폴더)를 만들어 정리할 수 있고, 만든 이미지·프롬프트는 **내 PC에 자동 저장**됩니다.

## ❓ 자주 묻는 것

- **돈 드나요?** — 본인 ChatGPT 구독 할당량을 쓰며, 별도 추가 결제·API 키는 없습니다.
- **PC마다 로그인해야 하나요?** — 네, PC마다 처음 1회 ChatGPT 로그인이 필요합니다(`setup.bat`이 안내).
- **내 그림이 외부로 나가나요?** — 아니요. 100% 내 PC에 저장됩니다.
